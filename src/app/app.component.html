DocX necesita instalar:

npm install --save docx
npm install file-saver --save

En tsconfig.json agregar en compilerOptions:
"noImplicitAny": false,

Importar libreria docx de la siguiente manera:
DocX trabaja por "Section" donde cada documento esta formado por varias secciones, cada seccion es una agrupacion de parrafos
import { Document, Packer, Paragraph, TextRun } from "docx";
A traves de "Packers" docx transforma el codigo a formato .docx
El objeto "Document" es donde se añade todo el contenido, se puede exportar como uno desee
El objeto "Paragraph" permite añadir parrafos al doc

Para guardar el documento es necesario importar "saveAs" de la siguiente manera:
import saveAs from 'file-saver';

Para usar imagenes se crea una instancia de "ImageRun":
new ImageRun({ data:("base64") , transformation: { width: 100, height: 100, }, }),

Enviar el archivo como descargable en un navegador

Retorna un string de base64
